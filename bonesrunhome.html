The server that runs a whole lot of nothing

bonesrunhome is an IBM x3550 M4 - it currently is sitting on my desk in my bedroom, so I have a lovely 1U space heater

I wound up with it as UCC (UWA's University Computer Club) was throwing out a bunch of them due to the unfortunate habit they had of kamikazing themselves. Thankfully there was one remaining that hadn't blown up and with a firmware update it should hopefully not blow up anytime soon :P

It currently has two Xeon E5-2697 v2's (24 total cores at 3.50GHz) and 384GB of RAM with Alma linux as it's operating system, however the end goal for this server is to buy a rack (I now have a rack! See myRack) and move it into my living room (sorry housemates), chuck in a number of SSD's for storage (I may also add a 10Gbps nic too) - and then I'll need to find something to run on it lol. I will probably turn this into my main server, replacing bonescallhome (due to this being a fair bit more powerful) and end up converting bonescallhome into a pfsense router.
