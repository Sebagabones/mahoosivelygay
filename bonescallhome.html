The possibly most sensible server I have

bonescallhome is a IBM x3200 M3, which is running Ubuntu server with 32GB of ram. A 4TB RAID-10 storage has been set up using older spinning HDDs, while the server boots off a 500GB boot SSD.

The end goal with this is to probably turn it into a router using pfsense, and move the website hosting onto bonesrunhome, as bonescallhome is a tower server I've got about 16 1Gbps RJ45 ports in the PCIE slots, although I'll probably change some of those to some 10Gbps (or possibly only 2.5Gbps lol)
